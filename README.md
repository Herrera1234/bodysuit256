### PROCESO DE INSTALACION
Instala los archivos necesarios
```bash
- termux-setup-storage
- pkg upgrade && update
- pkg install nodejs -y
- pkg install git -y
- pkg install bash -y
```

Clona este repositorio
 ```bash
> git clone https://github.com/thechoute/Baby-Ofc
```

Inicia la instalacion
```bash
- cd Baby-Ofc
- bash install.sh
```

Para Iniciar el Bot
 ```bash
- node thechoute
```
